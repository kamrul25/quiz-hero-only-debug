1. What is the difference among map(), filter(), find() and for Each?

Answer: They all are array method.Find method gives a single value. Map , filter and forEach method give a new array. Map and filter method return a new array and forEach return undefined. Map method executes the same code on every element in an array and returns a new array with the updated elements. ForEach is used to execute the same code on every element in an array but does not change the array. Filter checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria.

2. What is the difference local storage, session storage and cookies?

Answer: Local storage is useful for storing data that the user will need to access later, such as offline data. Local storage stores data with no expiration date. Session storage is a great way to improve the performance of your web applications. Session storage stores data for one session (data is lost when the browser tab is closed). Cookies are a good choice for storing data that should not be persisted for a long time. Cookies are small text files used to store user information on the user's computer.
